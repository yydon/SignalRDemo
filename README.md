# 使用
1. 開啟"Server", "Channel A"和"Channel B"連結(可分別用不同瀏覽器)。
2. 在"Server"頁面選擇要傳送訊息的頻道，輸入訊息後按下"傳送"。
3. 可即時於選擇的頻道頁面看到訊息。
